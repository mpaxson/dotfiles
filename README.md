# dotfiles

To Instlal this (if on ununtu 14.04 or higher) run 
```
Sudo apt-get update 
sudo apt-get install powerline 
sudo apt-get install toilet 
```

1. Navigate to `~/.local/lib/python2.7/site-packages/powerline/config_files/` and replace the files from the powerline folder in this repo.
2. Replace your ~/.profile and ~/.bashrc script with the .bashrc and .profile scripts in this repo. 
3. Rejoice and enjoy being a... 

## Issues
The font can have issues like overlapping  as pwoerline needs specific fonts set by the terminal. See  https://github.com/powerline/fonts and try to install these.
